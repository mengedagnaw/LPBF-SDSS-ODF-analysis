
# LPBF-SDSS-ODF-analysis

MATLAB/MTEX scripts and figures for orientation distribution function (ODF) analysis of LPBF-fabricated super duplex stainless steel.

## Overview

This repository contains ODF plotting outputs for ferrite (α-Fe) and austenite (γ-Fe) phases under different processing/heat-treatment conditions.

The ODF sections are shown at:

- φ₂ = 0°
- φ₂ = 45°
- φ₂ = 65°
